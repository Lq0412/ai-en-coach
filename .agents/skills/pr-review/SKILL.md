# PR review

Prioritize correctness, regressions, security, data loss, and missing tests. Report actionable findings with precise locations; do not manufacture findings. Separate merge blockers from suggestions.

Read [references/full-guide.md](references/full-guide.md) when the complete review checklist, reviewer policy, or merge conditions are required.
