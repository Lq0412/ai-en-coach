# AI design smell

Favor clear hierarchy, deliberate alignment, restrained color, useful whitespace, and product-specific visual choices. Flag nested cards, decorative gradients, excessive glow, arbitrary pills, weak typography, and ornamental icons only when visible in the actual design.

Read [references/full-guide.md](references/full-guide.md) for the full audit checklist and concrete remediation patterns.
