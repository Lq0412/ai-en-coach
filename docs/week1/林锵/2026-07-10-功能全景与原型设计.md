## 一、功能全景（按 milestone 排）

### P0 — MS1 交付（7/12-7/19，跑通 Demo 主链路）

| # | 功能 | 说明 |
|---|---|---|
| P0-1 | 场景选择（4 子场景） | 自我介绍、行为面试、项目深挖、基础技术问答，各带题库 |
| P0-2 | 简历上传 | PDF 提取全文进 LLM 上下文，3 个选择题兜底 |
| P0-3 | 练习模式切换 | 自由练习（可暂停重来 / 看提示）vs 模拟模式（全真一次过） |
| P0-4 | 语音对话主界面 | AI 提问 → 用户语音回答 → 转写展示 → AI 追问/推进 |
| P0-5 | 三层对话状态 | 开放对话 → 定向追问 → 收束推进，UI 有状态感知 |
| P0-6 | 结构化反馈报告 | 定位到句子 + 原句 vs 优化表达 + 各维度定性标签 + 下次建议 |
| P0-7 | 练习历史 | 所有练习记录，区分进行中/已完成，可继续/可查看报告 |

### P1 — MS2 交付（7/19-8/2，教练差异化）

| # | 功能 | 说明 |
|---|---|---|
| P1-1 | 跨 session 进步锚点 | "上次 STAR 缺失 → 这次完整"的定性对比，嵌入报告页 |
| P1-2 | 学习面板 / 进步追踪 | 弱项雷达图、练习统计、各维度进步轨迹 |
| P1-3 | 智能复练推荐 | 首页根据历史弱项推荐下次练习内容 |
| P1-4 | 表达收藏 / 好句本 | 对话中点星标收藏 AI 的优化表达，积累个人表达库 |
| P1-5 | 难度选择 | 温和（给提示多）/ 标准 / 严格（追问更深，不给提示） |
| P1-6 | 录音 / 网络状态指示 | 录音中、识别中、AI 回复中、网络不稳定等实时状态 |

### P2 — 后续里程碑（2 个月后或资源充足时）

| # | 功能 | 说明 |
|---|---|---|
| P2-1 | 多 Agent 扩展 | 留学面试官、职场英语等，复用内核 |
| P2-2 | 自定义 Agent | 用户自定角色和评分维度 |
| P2-3 | 完整统计仪表盘 | 练习时长趋势、弱项变化曲线、长期成长报告 |
| P2-4 | 付费 / 订阅体系 | 免费试用 + 付费增值 |
| P2-5 | 移动端适配 | 架构留 Gateway，但页面另做 |
| P2-6 | AI 动态出题 | 基于用户背景和弱项 AI 生成新题，替代人工题库 |

### 三层对话状态（状态指示器切换）

| 状态 | 指示器文字 | 颜色 | 触发条件 |
|---|---|---|---|
| 开放对话 | "面试中" | 🟢 绿色 | AI 提问后等待用户回答 |
| 定向追问 | "追问中" | 🟡 黄色 | 用户回答后 AI 深入追问 |
| 收束推进 | "推进中" | 🔵 蓝色 | 追问完成，进入下一题 |

### 特殊交互

| 场景 | UI 表现 |
|---|---|
| 用户卡住 > 5 秒 | 底部出现关键词提示，如 "试试以 'We measured this by...' 开头" |
| 用户跑题 | AI 文字提示 "Let me refocus — can you tell me more about..." |
| 网络不稳定 | 录音按钮变橙色 + "网络不稳定" 文字提示 |
| 识别失败 | "没听清，请再说一次" + 录音按钮恢复可用 |
| 练习完成（最后一道题） | "练习结束，正在生成反馈报告..." → 自动跳转反馈报告页 |

### 标注

- MS1 标签（绿色）：全部对话 UI + 状态指示器 + 操作区 + 卡顿提示
- MS2 标签（黄色）：⭐ 收藏按钮
