# 5.建议选型

 **Qwen 3.7 ASR**

因为它在英语陪练场景里综合更均衡：成本可控，后续可自建降本；效果适合中英混说、真实口语和上下文识别；延迟和并发能力适合实时对话；国内部署稳定性更好；接入可先 API 后私有化，使用路径清晰；同时更容易满足语音数据合规、数据不出境和未成年人保护等法律法规要求。
