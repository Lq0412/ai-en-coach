## 🆕 2026 年技术课程

### 从 0 构建一个 Mini Claude Code（TypeScript）

**链接**: [2026/01. 从0构建一个Mini Claude Code(TypeScript)](https://github.com/1024XEngineer/techcamp/tree/main/2026/01.%20%E4%BB%8E0%E6%9E%84%E5%BB%BA%E4%B8%80%E4%B8%AAMini%20Claude%20Code%28TypeScript%29)

**核心内容**:

- 为什么 Agent 能"做事情"，而 ChatBot 不能
- ReAct 循环架构：观察 → 思考 → 行动
- 最小 Agent Demo：**40 行核心 Loop** 实现天气查询
- 引入 Vercel AI SDK，拆解 Claude Code 结构
- 工程经验：上下文管理、安全防护、系统提示词架构

**关键公式**: `Agent = ReAct + Tools + UI`

**技术栈**: Bun + TypeScript + Vercel AI SDK

**工具设计原则**: 精简到 4 个核心工具（Read、Write/Edit、Bash、WebFetch）
> "工具别贪多。每多一个，模型负担就加重。"
