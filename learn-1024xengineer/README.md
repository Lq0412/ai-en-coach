## 背景

1024XEngineer（原名 1024 Bootcamp）是由七牛云发起的工程师实训营，核心理念是 **"Learning by Doing"**——在真实项目中成长，通过开源协作和架构思维培养 AI 时代的工程师。

目前已举办到第 4 期+，组织下共有 18 个仓库，涵盖 AI Coding Agent、桌面助手、实训课程等方向。
